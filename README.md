## Assignment Summary

This repository contains Assignment 1 for CSCN8020 Reinforcement Learning Programming. The work is completed in one Jupyter Notebook. It includes four problems: designing a pick-and-place robot as an MDP, performing two iterations of value iteration on a 2x2 gridworld, implementing value iteration and in-place value iteration on a 5x5 gridworld, and implementing off-policy Monte Carlo with importance sampling.

## How to Run

1. Clone this repository.
2. Open the folder in Visual Studio Code.
3. Create and activate a virtual environment.
4. Install the required libraries:

```bash
pip install -r requirements.txt
```

5. Open `CSCN8020_Assignment1.ipynb`.
6. Select the Python kernel.
7. Run all cells from top to bottom.

## Main Files

- `CSCN8020_Assignment1.ipynb`: Main notebook with explanations, code, outputs, and talking points.
- `src/environments.py`: GridWorld environment and reward/transition logic.
- `src/agents.py`: Value Iteration, In-Place Value Iteration, and Monte Carlo agents.
- `src/policies.py`: Random behavior policy and greedy target policy.
- `src/utils.py`: Logging, table, and plotting helper functions.
- `logs/sample_execution.log`: Sample log file generated by running the notebook.

## Assumptions

- Discount factor gamma is set to 0.9.
- The 5x5 gridworld goal state is `(4, 4)`.
- Grey states are `(2, 2)`, `(3, 0)`, and `(0, 4)`.
- Problem 3 uses four movement actions: right, down, left, and up.
- Monte Carlo uses a maximum step limit to avoid infinite episodes.
- The code uses object-oriented design to keep environment, agents, policies, and utilities separate.
